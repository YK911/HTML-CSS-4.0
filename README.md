# Про курс: Основи веб-розробки

Цей курс пропонує всебічне знайомство зі світом веб-розробки, зосереджуючись на
таких ключових темах, як **HTML**, **CSS** та **оптимізація продуктивності**
веб-сайтів.

### Чого ви навчитеся:

- **Ролі та інструменти:** Розуміння обов'язків сучасного розробника та робота з
  актуальним стеком технологій.
- **HTML5 & CSS3:** Фундаментальні техніки розмітки та стилізації для створення
  адаптивних веб-додатків.
- **Продуктивність:** Глибоке розуміння рендерингу сторінок та методів
  оптимізації швидкодії веб-додатків.

### Результат:

До завершення навчання ви опануєте навички створення візуально привабливих
інтерфейсів та навчитеся значно покращувати досвід користувачів (UX) завдяки
оптимізації технічних показників проєктів.
