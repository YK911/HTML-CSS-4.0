## Структура заняття:

- Підготовка проекту до роботи
- Структурна розмітка сторінки
- Верстка секції Hero та Benefits
- Верстка секції Ingredients та Made
- Верстка секції Products
- Верстка секції Bestsellers та Reviews

## Технічне завдання:

- HTML-файл називається `index.html`
- У корені проєкту є папка `images` із зображеннями
- У назвах файлів відсутня кирилиця, пробіли. Назви містять тільки літери, числа
  і слова англійською мовою
- Вихідний код відформатований за допомогою `Prettier`
- Всі зображення та текстовий контент взяті з макету
